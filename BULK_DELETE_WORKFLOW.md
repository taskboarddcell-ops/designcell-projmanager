## Common Workflows

### Workflow 1: Delete All Visible Tasks
```
1. Click "Select All" checkbox
2. Click "Delete Selected (X)"
3. Confirm
4. Done!
```

### Workflow 2: Delete Specific Tasks
```
1. Check task 1
2. Check task 2
3. Check task 3
4. Click "Delete Selected (3)"
5. Confirm
6. Done!
```

### Workflow 3: Delete All Except Some
```
1. Click "Select All" checkbox
2. Uncheck tasks to keep
3. Click "Delete Selected (X)"
4. Confirm
5. Done!
```

### Workflow 4: Use Filters First
```
1. Set filter (e.g., "Completed" status)
2. Click "Select All"
3. Click "Delete Selected (X)"
4. Confirm
5. All completed tasks deleted!
```

## Tips & Tricks

### Efficient Selection
- **Keyboard**: Tab to checkboxes, Space to toggle
- **Mouse**: Click checkbox or row
- **Bulk**: Use "Select All" then deselect exceptions

### Safe Deletion
- **Preview**: Review confirmation dialog carefully
- **Filters**: Use filters to narrow selection first
- **Start Small**: Test with 1-2 tasks first
- **Check Count**: Verify number in button matches intent

### Troubleshooting
- **Button not appearing**: Select at least one task
- **Can't select**: Ensure logged in as admin
- **Deletion failed**: Check console for errors
- **Wrong tasks**: Cancel and reselect
